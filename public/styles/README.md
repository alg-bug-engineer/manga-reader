# 风格参考图片说明

在这个目录中，你需要放置三种漫画风格的参考图片和预览图片。

## 目录结构

```
public/styles/
├── peach-preview.png         # 蜜桃灰灰风格预览图
├── peach-reference.png       # 蜜桃灰灰风格角色参考图
├── cat-preview.png           # 暴躁猫风格预览图
├── cat-reference.png         # 暴躁猫风格角色参考图
├── doraemon-preview.png      # 哆啦A梦风格预览图
└── doraemon-reference.png    # 哆啦A梦风格角色参考图
```

## 风格说明

### 1. 蜜桃灰灰 (Peach)
- **特点**: 可爱粉嫩，温馨治愈，圆润线条
- **配色**: 粉色、桃色、暖色调为主
- **风格**: 软萌、可爱、适合温馨故事
- **预览图**: peach-preview.png
- **角色参考**: peach-reference.png

### 2. 暴躁猫 (Cat)
- **特点**: 夸张搞笑，动感十足，表情丰富
- **配色**: 对比鲜明，色彩活跃
- **风格**: 搞笑、逗趣、适合喜剧风格
- **预览图**: cat-preview.png
- **角色参考**: cat-reference.png

### 3. 哆啦A梦 (Doraemon)
- **特点**: 简约友好，蓝白色调，经典怀旧
- **配色**: 蓝色、白色为主，清新简洁
- **风格**: 怀旧、友好、适合全年龄段
- **预览图**: doraemon-preview.png
- **角色参考**: doraemon-reference.png

## 如何添加图片

1. 准备对应风格的预览图和角色参考图
2. 将图片重命名为上述文件名
3. 放置在 `public/styles/` 目录下
4. 确保图片格式为 PNG 或 JPG

## 图片尺寸建议

- **预览图**: 400x400 像素或以上
- **参考图**: 512x512 像素或以上（用于AI生成参考）

## 注意事项

- 参考图会作为 Gemini API 生成图片时的角色形象参考
- 确保参考图中的角色形象清晰、特征明显
- 建议使用官方的角色形象或授权的图片
